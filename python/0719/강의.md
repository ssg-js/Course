## 프로그래밍

### -> 저장, 처리

| 저장   | 처리                  |
| ---- | ------------------- |
| 변수   | 연산자                 |
| 컨테이너 | 제어문(if, while, for) |
| 자료구조 | 알고리즘                |

### 컨테이너

- 시퀸스형 : 인덱스로 접근가능
  
  - 리스트 [] - mutable(가변형)
  
  - 튜플 () - immutable(불변형)
  
  - 레인지  range() - immutable

- 비시퀸스형
  
  - 세트 {} - mutable
  
  - 딕셔너리 {"key" : "value"} - mutable
