## 메서드

### dictionary.get()

- dic.get('key') == dic['key']  : 단, 해당 값이 존재하지 않을 경우 default 값이 None 반환

- dic.get('key', '여기에 입력하세요')  : None 값 대신 '여기에 입력하세요'를 반환시킬 수 있음
