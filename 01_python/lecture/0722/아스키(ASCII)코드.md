# 아스키(ASCII) 코드

## 아스키 코드

- **"American Standarad Code for Information Interchange" (미국정보교환표준부호)** 의 약자

- 컴퓨터는 숫자만 이해할 수 있다

| 비트(bit)          | 바이트(byte)                            |
| ---------------- | ------------------------------------ |
| 0과 1 두 가지 정보만 표현 | 데이터를 저장하는 기본  단위 <br>1 byte == 8 bit |

### 명령어

#### 1) ord(문자)

- 문자 -> 아스키코드로 변환하는 내장함수

#### 2) chr(숫자)

- 아스키코드 -> 문자로 변환하는 내장함수
